# Meu Primeiro Projeto

> Projeto de aprendizado de Git, GitHub e boas práticas de desenvolvimento.

---

## Descricao

Este projeto foi criado para aprender e praticar os fundamentos de versionamento de código com Git e GitHub, incluindo criação de repositórios, commits, branches e deploy.

---

## Tecnologias Usadas

- **Git** — Controle de versão
- **GitHub** — Hospedagem do repositório
- **Markdown** — Documentação

---

## Como Rodar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/meu-primeiro-projeto.git
   ```

2. Entre na pasta:
   ```bash
   cd meu-primeiro-projeto
   ```

3. Abra no VS Code:
   ```bash
   code .
   ```
